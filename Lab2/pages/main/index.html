Класифікація домашніх тварин

Класифікація домашніх тварин зазвичай базується на їхньому виді та родині в системі біологічної класифікації.

  1. Собаки (Canis lupus familiaris):

    • Клас: Мамалії (Mammalia)
    • Ряд: Хижі (Carnivora)
    • Родина: Псові (Canidae)
    • Вид: Собака (Canis lupus familiaris)

  2. Коти (Felis catus):

    • Клас: Мамалії (Mammalia)
    • Ряд: Хижі (Carnivora)
    • Родина: Котові (Felidae)
    • Вид: Кіт домашній (Felis catus)

  3. Кролики (Oryctolagus cuniculus):

    • Клас: Мамалії (Mammalia)
    • Ряд: Гризуни (Rodentia)
    • Родина: Зайцеві (Leporidae)
    • Вид: Кролик домашній (Oryctolagus cuniculus)

  4. Прибережні свині (Sus scrofa domesticus):

    • Клас: Мамалії (Mammalia)
    • Ряд: Копитні (Artiodactyla)
    • Родина: Свиневі (Suidae)
    • Вид: Прибережна свиня (Sus scrofa domesticus)

  5. Коні (Equus ferus caballus):

    • Клас: Мамалії (Mammalia)
    • Ряд: Непарнокопитні (Perissodactyla)
    • Родина: Коневі (Equidae)
    • Вид: Кінь (Equus ferus caballus)
